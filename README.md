# Game of Khan

The Game of Khan project is an implementation of a game artificial intelligence in Prolog.
This projet was done in the context of the IA02 course, at the University of Technology of Compi√®gne.

## Running the application

Install the GNU Prolog interpretor:

```
sudo apt-get install gprolog
```

And run the following command :

```
gprolog
[main],main(X).
```
